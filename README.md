# 원광대학교 소개 & 홍보 서비스

학교 홈페이지의 캠퍼스 정보는 친절하지 않고 흥미를 끌지 않아 사람들이 잘 찾아보지 않는다. <br>
이런 점을 보완하고자 사람들의 '호기심'과 '흥미'를 유발하여 **보고 싶도록 만드는 학교 소개 서비스**를 만들게 되었다.

<br>

👉 [서비스 링크](https://wkuintro.netlify.app/)
